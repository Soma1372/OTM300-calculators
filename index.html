2. Intro to Operations

Little’s Law

Formula: I = R × T, where I is average inventory (flow units), R is average flow rate (flow units per time), and T is average flow time (time). Little’s Law describes average values.

6. Process Interruptions

Formulas (as written in slides):
Batch size = (Target capacity × Total setup time) / (1 − (Target capacity × Processing time))

Batch Size

Batch size is the number of units produced per setup (units). Target capacity is units/time. Setup time is time. Processing time is time/unit.
