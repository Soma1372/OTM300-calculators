10. Forecasting
Use a hybrid setup: simple formulas are listed for the Enhanced General Calculator, while multi-step forecasting methods have dedicated input panels below.

Formula-Only Items (Use the General Calculator)

These are intentionally left as formulas so students still identify the correct setup.

Forecast Error (FEt) = Forecast − Actual = ŷt − yt Squared Error = (FEt)^2 Absolute Error = abs(FEt) MSE = (sum of squared errors) / n MAE = (sum of absolute errors) / n Naïve Forecast = last realized value = yt Moving Average Forecast = (sum of last T realized values) / T Forecast Combination = (sum of forecasts) / n
Example expressions students can type above:
100-110
abs(100-110)
((-10)^2)
(64+49+81)/3
(10+14+21)/3
